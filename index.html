How It Works · 怎麼運作

鍵少、鍵大

37 個注音依語音分組收進 12 大鍵。國語實際只用 412 個音節，不存在的組合直接排除。每個鍵都大到不會按錯隔壁。

點擊 + 四方向 flick

靈感來自日文的 flick 鍵盤——點一下是中央、往不同方向滑就是同組其他注音。介音可拖動到韻母滑動條，一個動作完成整組韻母。每個手勢都有確定的結果。

三區分明 + 智慧灰化

聲母在左、介音在中、韻母在右。打了聲母後，不可能接的組合會變灰——就像導航軟體把封閉的路標灰色。位置不變，只是不需要的路淡出。

Advanced · 進階體驗

中英混打

同一個鍵盤打中英；學習模式按鍵會顯示對應字母提示 — 例：ㄛㄎ → OK、ㄒㄍㄨㄤ → Ｘ光。

Privacy · 隱私

不存取網路

不收集打字內容

Full Access 透明註解

iOS 鍵盤要呼叫震動 API 必須開啟 Allow Full Access — 這是 Apple 平台限制。胖打不論 Full Access 開關都不傳任何資料出去，上面兩條事實獨立成立。

學習資料留在裝置本機

自動學習的字/詞只留在你裝置（如開 iCloud Sync 則同步至你自己的 iCloud 私人空間，不經 panda-zhuyin server）；可用「忘記」功能個別清除或一鍵重置。

Features · 完整功能

核心（永遠免費）

  • 滑動注音輸入

    37 注音收進 12 大鍵 + flick

  • 自訂字典

    加入你自己的字詞。

  • 組字引擎

    Megrez 開源引擎，依你打的注音給出對的字。

  • 自動學習

    記住你的選字選擇 — 事實，非預測。

  • Emoji 候選

    「貓熊」→ 🐼

  • 多字前綴聯想

    「你好」→ 嗎 / 呀 …

  • 中文數字 / 日期時間 parser

    千分位、逐字、萬進位、日期時間；漢字大寫 壹/貳/參 從候選區取用。

  • 候選分類 tab

    詞頻 / 符號 / 字母 三 tab。

  • 擴充注音符號

    老國音、方音；設定可開。

  • 空白鍵水平移動游標

    長按空白鍵橫向移動游標。

  • 符號字典疊加

    「箭頭」→ →、「攝氏」→ ℃ — iOS 原生僅在漢字輸入時支援 emoji 替換。

  • 框選替換候選

    選文字 → 候選列出現替換選項。

附加 / 進階

  • iCloud Sync

    多裝置共享自訂詞與學習記憶（採用者選擇開啟）。
