How It Works · 怎麼運作
  1. 01

    大鍵分組

    注音按發音分組合併，國語只用 412 個音節，多餘組合排除。每個鍵都大到不會誤觸。

  2. 02

    每個手勢一個音

    每個鍵都是「中央 + 四個方向」，點哪邊滑哪邊都對應一個確定的音。介音長按展開選盤，順手再滑一下就把韻母也選好，整組音節一次完成。

  3. 03

    三區分明 + 智慧灰化

    聲母在左、介音在中、韻母在右。打了聲母後，不可能接的組合會變灰——就像導航軟體把封閉的路標灰色。位置不變，只是不需要的路淡出。

Advanced · 進階體驗

中英混打

學習模式

同一個鍵盤打中英；學習模式按鍵會顯示對應字母提示 — 例：ㄛㄎ → OK、ㄒㄍㄨㄤ → Ｘ光。

「忘記」功能

學習資料控制

個別忘記某個學過的字／一鍵重置所有學過的詞 — 你掌控你的學習資料。

進階符號輸入

符號鍵盤

符號鍵盤含 Mozc 衍生變體（paired 括號 4 組樣式、連點循環、貨幣/運算/IP/排版分類）；可選擇開啟擴充注音符號（老國音、方音、ㄮ 等）。

數字鍵下滑漢字數字

Flick 輸入

數字鍵往下滑直接輸入漢字小寫一/二/三/四 — 不用先打阿拉伯數字再從候選挑。

Privacy · 隱私

不存取網路

胖打注音不會主動連線到外部網路，鍵盤完全離線運作。

不收集打字內容

你的輸入內容不會離開裝置 — 不會傳給我們、不會傳給任何人。

Full Access 透明註解

iOS 鍵盤要呼叫震動 API 必須開啟 Allow Full Access — 這是 Apple 平台限制。胖打不論 Full Access 開關都不傳任何資料出去，上面兩條事實獨立成立。

學習資料留在裝置本機

自動學習的字/詞只留在你裝置（如開 iCloud Sync 則同步至你自己的 iCloud 私人空間，不經 panda-zhuyin server）；可用「忘記」功能個別清除或一鍵重置。

Features · 完整功能

注音輸入

滑動注音為核心：37 個注音收進少量大鍵，四方向 flick 對應同組音節一氣呵成。Megrez 開源組字引擎把輸入變成對的字。可選擇開啟擴充注音（老國音、方音、ㄮ 等）作學術或方言使用。長按空白鍵橫向移動游標。

  • 大鍵 + flick 整音節輸入
  • Megrez 組字引擎
  • 擴充注音符號（設定可開）
  • 空白鍵長按橫向游標

字典學習

自訂字典加入你自己的字詞；自動學習記住你的選字選擇（事實，非預測）；多字前綴聯想自動補：「你好」→ 嗎 / 呀。如開啟 iCloud Sync 則跨裝置同步至你自己的 iCloud 私人空間，不經 panda-zhuyin server。

  • 自訂字典
  • 自動學習（事實，非預測）
  • 多字前綴聯想
  • iCloud Sync（可選開啟）

介面體驗

候選列分三個 tab：詞頻 / 符號 / 字母。Emoji 候選（「貓熊」→ 🐼）；符號字典疊加（「箭頭」→ →、「攝氏」→ ℃）— iOS 原生僅在漢字輸入時支援 emoji 替換。中文數字 / 日期時間 parser 處理千分位、萬進位、日期時間，漢字大寫壹/貳/參 從候選區取用。選取既有文字時，替換選項會出現在候選列。

  • 候選 tab：詞頻 / 符號 / 字母
  • Emoji 候選
  • 符號字典疊加
  • 中文數字 / 日期時間 parser
  • 框選替換候選
