隱私權政策

胖打注音（Panda Zhuyin）的隱私權政策

概述

胖打注音是一款注重隱私的 iOS 注音輸入法鍵盤。我們的核心原則是：你的打字內容只屬於你自己。

兩條核心事實

胖打注音的運作建立於兩條獨立成立的事實：

  • 不存取網路 — 鍵盤程式碼中沒有任何網路請求。
  • 不收集打字內容 — 你的輸入內容不會離開你的裝置。

不包含任何分析工具、追蹤器或第三方 SDK。

Full Access 透明註解

胖打注音需要「允許完整取用」權限，唯一的原因是按鍵震動回饋。iOS 鍵盤 extension 必須開啟 Full Access 才能呼叫震動 API — 這是 Apple 平台限制，與資料隱私無關。

開啟完整取用後，胖打注音仍然：

  • 不存取網路 — 鍵盤程式碼中沒有任何網路請求。
  • 不傳送任何資料 — 你的輸入內容不會離開你的裝置。

上面兩條事實獨立於 Full Access 開關成立。

學習資料留在裝置本機

自動學習的字／詞只留在你裝置（如開啟 iCloud Sync 則同步至你自己的 iCloud 私人空間，不經 panda-zhuyin server）；可用「忘記」功能個別清除或一鍵重置。

完全離線運作

所有功能——包括注音輸入、選字、音韻判斷——全部在你的裝置上本地運行。無論你是否連接 Wi-Fi 或行動數據，都不影響使用。

Host App

胖打注音的主程式（Host App）僅包含操作教學和鍵盤啟用引導，不收集任何使用者資料。

兒童隱私

胖打注音不收集任何使用者的個人資訊，包括兒童在內。

政策變更
